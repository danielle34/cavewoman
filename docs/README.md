# Project site

Single-page CAVEWOMAN site, built on the Nerfies template (CC BY-SA 4.0).

## Files

```
site/
├── index.html
├── DEPLOYMENT.md
├── .nojekyll
└── static/{css,js,images}/
```

CDN deps: Bulma 0.9.4, Font Awesome 5.15.4, Academic Icons, Google Fonts.

## Local preview

```bash
cd site && python3 -m http.server 8000
```
